# Nakiri Cafe ☕

Веб-сайт городского кафе «Nakiri» в Бишкеке. Курсовая работа по дисциплине «Веб-разработка».

## 🌐 Демо

После настройки GitHub Pages сайт будет доступен по адресу:
**https://tegatm.github.io/nakiri/**

## ✨ Что внутри

- **5 страниц**: главная, меню, доставка, контакты, о нас
- **36 позиций меню** в 8 категориях (кофе, лимонады, чай, завтраки, сэндвичи, салаты, десерты, выпечка)
- **Корзина** с сохранением между сессиями (localStorage)
- **Форма доставки** с расчётом стоимости
- **Бронирование стола**
- **Переключатель языка** RU / EN на всех страницах
- **Тёмная тема** с золотыми акцентами
- **Адаптивная вёрстка** — корректная работа на телефонах, планшетах и компьютерах

## 🛠️ Технологии

- HTML5, CSS3, JavaScript (ES6+)
- CSS Grid и Flexbox для адаптивных макетов
- JSON для хранения каталога
- localStorage для корзины и заказов
- Шрифты Playfair Display и Inter

## 📁 Структура

```
.
```
